## วิธีรันโปรเจกต์

1. โหลด Mysql Workbench มา
2. สร้าง database ใน MySQL ชื่อว่า `canteen_db`
3. copy ไฟล์ `application.properties.example` 
   แล้วเปลี่ยนชื่อเป็น `application.properties`
4. แก้ username และ password ให้ตรงกับเครื่องตัวเอง
5. รัน Spring Boot ได้เลย — Hibernate จะสร้าง Table ให้อัตโนมัติ
